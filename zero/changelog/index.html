2016-01-11

Versions

  • Printix Client for Windows version 1.1.408.0.
  • Printix Client for Mac version 1.1.236.0.

Known issues

  • Printix Client on Windows 7 (32-bit and 64-bit) may prevent sign in and complain about missing System.web.dll and errors in production.js. This happens if Microsoft .NET Framework 3.5 is installed. To resolve upgrade to Microsoft .NET Framework 4.
  • Translations are made with the use of Google Translate. Please inform us about improvements for translations.
